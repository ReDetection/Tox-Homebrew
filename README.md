Homebrew formulae to install Tox.  

**Tap it**:

    brew tap Tox/tox

**Install core**:

    brew install --HEAD libtoxcore

**Install toxic**:

    brew install --HEAD toxic

**Install venom**:

    brew  install --HEAD venom

**Install Poison**:

    brew  install poison
    # OR...
    brew  install --HEAD poison # unreliable and dangerous!
